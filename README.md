
## 🎬 Movie Info Card

A simple and responsive Movie Info Card web application built using **Node.js**, **Express.js**, and **EJS**. This project displays movie information with an attractive user interface and demonstrates server-side rendering using Express and EJS templates.

## 🚀 Features

- 🎥 Display movie information
- 📱 Responsive user interface
- ⚡ Server-side rendering with EJS
- 🎨 Clean and simple design
- 📂 Organized project structure

## 🛠️ Technologies Used

- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript

## 📁 Project Structure

```
movie-info-card/
│── node_modules/
│── public/
│── views/
│── server.js
│── package.json
│── package-lock.json
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/movie-info-card.git
```

2. Navigate to the project folder

```bash
cd movie-info-card
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
node server.js
```

or

```bash
npm start
```

5. Open your browser and visit

```
http://localhost:3000
```
